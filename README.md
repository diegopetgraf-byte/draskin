# Esthétique - Papelaria Veterinária Personalizada

Managed eCommerce codebase for Esthétique.

## Tech Stack
- React / Vite
- TypeScript
- Tailwind CSS / Shadcn UI
- Supabase (Database & Edge Functions)
- Hosted on Vercel

## Development

```sh
npm install
npm run dev
```

## Production
Code is automatically deployed to Vercel on push to `main`.
